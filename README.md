# Fun_Corp_data_analytcs_steam
Analise de dados de uma empresa de jogos de tabuleiro, para avaliar o mercado para um possivel lançamento de um novo jogo
Atualmente, a empresa opera somente com produtos físicos, como jogos de tabuleiro e estratégia e action figures colecionáveis. Contudo, os investidores desejam expandir as operações da empresa para o mercado de jogos digitais. Para montar uma estratégia de expansão de mercado, a Fun Corp. realizou uma coleta de dados na Steam, uma das maiores plataformas de distribuição de jogos digitais do mundo. Estes dados seriam utilizados para deteção de tendências e padrões de consumo e identificação de competidores relevantes no mercado. 

Conjunto de Dados

A Steam é uma plataforma de jogos criada pela empresa Valve. Esta plataforma disponibiliza diversas funcionalidades, como gestão de direitos digitais (DRM), servidores de jogos, solução de anti-cheat, rede social, serviços de streaming, entre outros. O conjunto de dados utilizado no desenvolvimento deste trabalho foi criado utilizando as APIs disponibilizadas pelas plataformas Steam e Steam Spy.
